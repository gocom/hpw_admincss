h1. hpw_admincss

"Download":https://github.com/gocom/hpw_admincss/releases | "Packagist":https://packagist.org/packages/rah/hpw_admincss | "Donate":http://rahforum.biz/donate/hpw_admincss

Fork of "hpw_admincss":http://textpattern.org/plugins/434/hpw_admincss. This plugin loads style sheet *admin* on your Admin side pages. No extra bells or whistles. Once installed, create a style named *admin* via the Styles panel and start hacking away.

h2. Install

Using "Composer":http://getcomposer.org:

bc. $ composer require rah/hpw_admincss:*

Or "download":https://github.com/gocom/hpw_admincss/releases a plugin package.

h2. Requirements

* Textpattern 4.4.1 or newer.
* PHP 5.2.0 or newer.

h2. Toolshed notice

This is a toolshed project. Experimental and not part of the main supported product line of Rah; merely just a patch-job.
